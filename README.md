## ファイル構成

```
.
├── index.html          # 採用ページ本体（HTML/CSS/JSすべて1ファイル）
├── images/             # 画像アセット
│   ├── doteppan_logo.png    # 筆文字ロゴ
│   ├── do_mark.png          # 「ど」マーク
│   ├── monja_cooking.jpg    # もんじゃ調理シーン
│   ├── store_exterior.jpg   # 店舗外観
│   ├── staff_sapporo.jpg    # 札幌スタッフ

│   ├── staff_tsukishima.jpg # 月島スタッフ
│   └── staff_karaoke.jpg    # OFF-DUTYカラオケ
└── README.md
```
